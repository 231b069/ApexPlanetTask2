# ApexPlanet Task 2

## Overview
This project was developed as part of the ApexPlanet Web Development Internship Task 2.

The project demonstrates intermediate HTML, CSS, and JavaScript concepts including:

- Contact Form Creation
- JavaScript Form Validation
- Responsive Design using Flexbox and CSS Grid
- Dynamic To-Do List using DOM Manipulation

---

## Features

### 1. Contact Form
- Name Field
- Email Field
- Message Field
- Submit Button

### 2. Form Validation
- Required field validation
- Email format validation
- Error messages for invalid inputs
- Success message after valid submission

### 3. Responsive Layout
- Responsive Navigation Bar
- CSS Grid Layout
- Mobile-friendly design using Media Queries

### 4. Dynamic To-Do List
- Add new tasks
- Delete existing tasks
- Real-time DOM manipulation using JavaScript

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
